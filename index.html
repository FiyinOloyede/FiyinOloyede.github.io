Portfolio

Product Recommendation System

This project builds a recommendation engine that learns product–product relationships from real purchase baskets using Word2Vec. The pipeline, implemented in Python with pandas and NumPy for preprocessing and Matplotlib for validation, generates personalized suggestions from a UK online retail dataset (Dec 2010–2011). The outcome demonstrates a scalable approach to automate cross-sell logic and improve customer experience and retention.

House Price Prediction

This project analyzes King County home sales (May 2014–2015) using a PySpark MLlib pipeline to engineer features and evaluate multiple regressors. Model selection is driven by RMSE and R² on a hold-out set, providing a robust, scalable workflow for valuation scenarios and pricing sensitivity analysis across large datasets.

Predictive Analysis on Space Rocket Data

This project ingests launch data from REST APIs and web scraping, performs SQL-driven EDA, and maps outcomes with Folium. A Decision Tree classifier achieves approximately 86.2% accuracy for landing success, illustrating how interpretable features and transparent models can reduce uncertainty in mission planning and resource allocation.

Cyclistic Bike Share — Case Study

This case study explores behavioral differences between casual riders and members across a full year of trips. Using the tidyverse (dplyr, ggplot2), the analysis profiles usage by time, station, and trip length, surfacing high-leverage segments and time windows for targeted campaigns that convert casual riders to members.

Udemy Course Dashboard

This dashboard provides a single view of revenue, subscriber growth, and content performance from 2012 to 2017. A star-schema model with DAX measures supports NPS tracking, growth metrics, cohort analysis, and category drill-throughs, enabling pricing and content decisions grounded in stable KPIs and trend visibility.

Cribie User Engagement Dashboard

This dashboard tracks 2023 registrations, active users, and order volume by month and region. KPI scorecards, hierarchical drilldowns, and regional comparisons reveal growth pockets and underperforming areas, guiding targeted activation strategies and operational improvements.

Passenger Satisfaction Dashboard (Power BI & R)

This dashboard combines R-based EDA (tidyverse) with a Power BI data model and advanced DAX to analyze NPS and service attributes. The results isolate the levers most correlated with satisfaction uplift and present a clear, operational roadmap for improvement across customer touchpoints.
